                         GETTING HELP
═══════════════════════════════════════════════════════════════════════════════

Problem:            Solution:
──────────────────────────────────────────────────────────────────────────────
API Connection      1. Check backend running: curl localhost:8000
                    2. Verify .env.local has correct API URL
                    3. Check browser console (F12) for errors

Port Already Used   1. Use different port: npm run dev -- -p 3001
                    2. Kill process on port: lsof -ti:3000 | xargs kill

Build Error         1. Delete node_modules: rm -rf node_modules
                    2. Reinstall: npm install
                    3. Rebuild: npm run build

Session Lost        1. localStorage persists per browser
                    2. Clear cache to reset
                    3. Use "New Chat" to start over
